### 使用

+ 脚本`install.sh`用于将项目中的配置覆盖到本地上
+ 脚本`backup.sh`用于为本地的配置做备份
+ 脚本`collect.sh`用于将本地的配置归档回项目中

1. 按照我的教程配置Linux，即下载好zsh、oh-my-zsh和两个插件
2. 将该项目clone到本地并进入
3. 运行项目下的`install.sh`
